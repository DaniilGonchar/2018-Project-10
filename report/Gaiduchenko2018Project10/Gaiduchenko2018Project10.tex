\abstract
    {В данной работе рассматривается задача  обнаружения текста на изображениях. Приведён сравнительный анализ state-of-the-art  архитектур для обнаружения текста и морфологических методов обучения без учителя. Предложен алгоритм, основанный на использовании как нейросетевых, так и морфологических методов в зависимости от специфики задачи. 
    
\bigskip
\textbf{Ключевые слова}: \emph{нейронные сети, непрерывно-морфологические методы, распознование текста, распознование изображений обнаружение текста на изображении, детекция текста, морфологические методы, нейронные сети для обнаружения текста, обучение без учителя, анализ текстов}.}
\titleEng
    {Comparison of neural network and continuous-morphological methods in the task of text detection.}
\authorEng
    {Gaiduchenko~N.\,E., Akimow~K.\,M., Mironova~L.\,R. et al.}
%\organizationEng
%    {$^1$Organization; $^2$Organization}
\abstractEng
    {This paper deals with the problem of finding text on images. A comparative analysis of state-of-the-art architectures for text detection and morphological unsupervised learning methods is given. An algorithm based on the use of both neural network and morphological methods, depending on the specifics of the problem, is proposed.

\bigskip
\textbf{Keywords}: \emph{neural networks, continuous morphological methods, text detection, computer vision algorithms, image recognition, morphological methods, neural networks for text detection, unsupervised learning, text analysis}.}
